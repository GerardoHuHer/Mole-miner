# Proyecto Final - Videojuego en Godot

Este repositorio contiene el desarrollo de un videojuego realizado en Godot Engine como parte del proyecto final.

El proyecto incluye escenas, recursos gráficos, scripts de lógica y archivos de configuración necesarios para su ejecución.

## Estructura del proyecto

- `assets/` → Recursos gráficos y elementos visuales
- `scenes/` → Escenas principales del videojuego
- `player.gd` → Script principal del jugador
- `project.godot` → Archivo de configuración del proyecto

## Requisitos

Para ejecutar este proyecto es necesario tener instalado:

- Godot Engine 4.x

## Cómo ejecutar el proyecto

1. Clonar este repositorio:

```bash
git clone https://github.com/GerardoHuHer/Mole-miner.git

Abrir Godot Engine.
Seleccionar la opción Import.
Buscar la carpeta del proyecto clonado y seleccionar el archivo:
project.godot
Importar el proyecto y abrirlo.
Ejecutar el proyecto presionando el botón Run dentro de Godot.
